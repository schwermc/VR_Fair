# VR_Fair
A simple game of Kingyo-sukui. <br />

**Game Play** <br />
Grab a *poi* to catch fish to earn tickets! <br />

**Controls** <br />
Use left stick to move <br />
Use right stick to rotate <br />
Press A to crouch and uncrouch <br />
Press trigger to grab <br />

**Assest** <br />
Models are self-made <br />
